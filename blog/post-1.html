如何搭建一个零成本的个人网站

很多人以为做个人网站又贵又麻烦，其实用纯静态页面 + GitHub Pages，全程可以做到零成本、零服务器。这篇就带你走一遍。

第一步：写页面

一个 index.html 加上 CSS/JS 就够了。不需要框架、不需要构建。把内容填进去，本地双击打开就能预览。

第二步：推到 GitHub

新建一个仓库（建议用 你的用户名.github.io 作为仓库名，这样可以直接当主站），把文件推上去：

git init
git add .
git commit -m "initial site"
git branch -M main
git remote add origin git@github.com:你的用户名/你的用户名.github.io.git
git push -u origin main

第三步：开启 Pages

在仓库 Settings → Pages 里，Source 选择 main 分支、目录选 / (root)，保存后等一两分钟，访问 https://你的用户名.github.io 就能看到网站了。

小提示：仓库根目录放一个空的 .nojekyll 文件，可以避免 GitHub 用 Jekyll 处理你的下划线文件夹。

小结

这就是最小可行路径。后面想加博客、换主题、绑定自定义域名，都是在这个骨架上慢慢长出来的事。
